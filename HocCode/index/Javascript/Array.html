Array Methods Cheat Sheet

Thuộc tính / Hàm Công dụng Ví dụ ✓
lengthĐộ dài mảng[1,2,3].length // 3
push(x)Thêm phần tử cuối mảngarr.push(4)
pop()Xóa & trả về phần tử cuốiarr.pop()
shift()Xóa & trả về phần tử đầu tiênarr.shift()
unshift(x)Thêm phần tử vào đầu mảngarr.unshift(0)
indexOf(x)Vị trí phần tử đầu tiên["a","b"].indexOf("b") // 1
lastIndexOf(x)Vị trí phần tử cuối cùng["a","b","a"].lastIndexOf("a") // 2
includes(x)Kiểm tra có tồn tại[1,2,3].includes(2) // true
join(",")Nối thành chuỗi[1,2,3].join("-") // "1-2-3"
slice(a,b)Cắt mảng (a → b-1)[1,2,3,4].slice(1,3) // [2,3]
splice(a,n,...)Xóa/thêm phần tử từ vị trí aarr.splice(1,2)
concat(arr2)Ghép mảng[1,2].concat([3,4])
reverse()Đảo ngược[1,2,3].reverse()
sort()Sắp xếp (mặc định string)[3,1,2].sort()
map(fn)Tạo mảng mới bằng hàm[1,2,3].map(x=>x*2)
filter(fn)Lọc theo điều kiện[1,2,3,4].filter(x=>x>2)
reduce(fn,init)Tính gộp[1,2,3].reduce((a,b)=>a+b,0)
forEach(fn)Lặp mảng (ko trả về mới)[1,2,3].forEach(x=>console.log(x))
