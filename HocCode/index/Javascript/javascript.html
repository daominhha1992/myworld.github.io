Tổng Hợp Kiến Thức JavaScript

Keyword (từ khóa) → ngôn ngữ JS định sẵn (như let, const, if).
Identifier (định danh) → tên bạn tự đặt cho biến, hàm, class (vd: register, errorMsg).
Property (thuộc tính) → giá trị hoặc đặc điểm của object (vd: .textContent).
Method (phương thức) → hàm có sẵn trong object (vd: document.getElementById).
