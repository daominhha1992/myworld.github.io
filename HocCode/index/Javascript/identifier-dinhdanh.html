📌 Identifier (Định danh) trong JavaScript

Loại Identifier Ví dụ Mô tả Đã học
Biến (Variable) let userName = "Minh"; Tên biến do lập trình viên đặt
Hàm (Function) function registerUser() { ... } Tên hàm do người dùng định nghĩa
Lớp (Class) class UserAccount { } Tên class do lập trình viên đặt
Đối tượng (Object) const person = { name: "Minh" }; Tên object (biến chứa object)
Property person.age = 20; Tên thuộc tính trong object cũng là identifier
Tham số (Parameter) function sum(a, b) { return a + b; } Tên tham số trong định nghĩa hàm
Hằng số (Constant) const MAX_USERS = 100; Tên hằng số (theo convention thường viết HOA)
