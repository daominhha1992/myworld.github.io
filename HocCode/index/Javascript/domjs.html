DOM Methods & Properties Cheat Sheet

Category Method / Property Description ✔
Truy xuất phần tử getElementById(id) Lấy phần tử duy nhất theo id
getElementsByClassName(className) Lấy tất cả phần tử có class (HTMLCollection)
getElementsByTagName(tagName) Lấy tất cả phần tử theo tag
querySelector(selector) Lấy phần tử đầu tiên khớp selector CSS
querySelectorAll(selector) Lấy tất cả phần tử khớp selector CSS (NodeList)
Nội dung textContent Nội dung text của node
innerHTML Nội dung HTML của phần tử
outerHTML Nội dung HTML bao gồm chính phần tử
innerText Nội dung text hiển thị (theo CSS)
Style / Class element.style.property Thay đổi style trực tiếp (inline)
element.classList.add/remove/toggle Thêm, xóa, hoặc chuyển đổi class
element.className Thay đổi toàn bộ class
Thêm / Xóa phần tử appendChild(node) Thêm node con vào cuối phần tử
insertBefore(newNode, referenceNode) Chèn node trước node tham chiếu
removeChild(node) Xóa node con
replaceChild(newNode, oldNode) Thay node cũ bằng node mới
cloneNode(true/false) Sao chép node (true = deep copy)
Duyệt DOM parentNode Node cha
childNodes Tất cả node con (bao gồm text nodes)
children Chỉ các phần tử con (element nodes)
firstChild / lastChild Node con đầu/cuối
nextSibling / previousSibling Node kế / trước node hiện tại
firstElementChild / lastElementChild Node element đầu/cuối
nextElementSibling / previousElementSibling Element kế / trước node hiện tại
Event addEventListener(event, callback) Lắng nghe sự kiện
removeEventListener(event, callback) Xóa listener sự kiện
Tạo phần tử document.createElement(tagName) Tạo element mới
document.createTextNode(text) Tạo node text mới
Thuộc tính khác id Lấy/đặt id
className Lấy/đặt class
value Lấy/đặt giá trị input, textarea, select
dataset Lấy/đặt data-* attributes
