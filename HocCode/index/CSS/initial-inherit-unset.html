Phân biệt inherit, initial, unset

Đây là phần tử cha. Màu chữ là Blue, Viền là Green.

1. Thuộc tính Color (Được kế thừa)

color: inherit; (Kết quả: Blue) color: initial; (Kết quả: Black) color: unset; (Kết quả: Blue, hoạt động như inherit)

2. Thuộc tính Border (KHÔNG được kế thừa)

border: inherit; (Kết quả: Viền Green) border: initial; (Kết quả: Viền none) border: unset; (Kết quả: Viền none, hoạt động như initial)
